# ChatMat Frontend (Vite + Vue 3)

Quick start (macOS, bash):

1. Install dependencies

```bash
cd frontend
npm install
```

2. Run dev server

```bash
npm run dev
```

The Vite dev server runs on http://localhost:5173 and proxies `/api/*` to http://127.0.0.1:8000. The chat UI includes a floating input aligned to the left chat column.
